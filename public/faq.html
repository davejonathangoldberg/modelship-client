FAQ

What is ModelShip?

Design, build and deploy APIs without coding

ModelShip is a tool that lets you design, build and deploy APIs without coding. It lets you focus on the key concepts you want in your application, and it automatically applies best-practices in API design so you don't have to. In short, you tell us about the things in your application, and how they relate to each other, and ModelShip will do the rest.

What is 'the rest', you might ask? ModelShip will take your design, and create a series of API endpoints that conform to modern REST API design principles and best-practices. We also automatically generate a Swagger file that you can use for interactive documentation of your API, and we deploy your API to the cloud so you can begin using it immediately.

Where are my APIs deployed to?

Independence is happiness.

ModelShip currently deploys APIs to Heroku and DigitalOcean. However, the design of ModelShip allows for deployment of an API to any PAAS or IAAS system. Avoiding lock-in and retaining independence and interoperability are key tenets of ModelShip's design.

How long will my API be available?

Use your API free for 7 days

When you create an API using ModelShip, it is available freely for 7 days. You'll always have the option to deploy it directly to a cloud service for a small one-time fee (Heroku is currently being supported, Digital Ocean is on its way). You will also have the option to download the code of your API and deploy it yourself. The download option will be free while ModelShip is in early beta.

Can I use my API in production?

This is not recommended...yet

Currently ModelShip deploys APIs that work over HTTP, not HTTPS. All communication from client to API is unencrypted, and so it is not recommended to use your ModelShip API in production. If you download the code to your API, you can add an SSL certificate and quickly convert it to work over HTTPS. We have plans to add SSL to every API we deploy, but this feature is not ready yet!

Today, ModelShip is great for prototyping and quickly adding a simple API for your applications to use. Please let us know how you would like to use it!

Can I download the code for my API?

Yes!

At ModelShip, we are anti lock-in. We want to make it easy for you to create APIs and applications and deploy them wherever you want. While ModelShip is in beta, we are happy to send you a link to download your code for free. We plan to charge a small fee for access to code repositories when we come out of beta.
